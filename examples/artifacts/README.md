# Example Artifacts

This directory is for curated example outputs only.

## What these are

Files here are historical snapshots of generated validation artifacts. They show the shape and schema of outputs produced by Qnty's research workflows.

## What these are NOT

- These are **not** live trading approval.
- These are **not** proof of deployable alpha.
- These are **not** current or guaranteed to reproduce identically, because market data changes.

## On labels like GO / PASSED / SURVIVED

If an example artifact contains labels such as `verdict: GO`, `verdict: PASSED`, or `SURVIVED`, they mean only:

- **GO** → "Continue research / not killed by this test."
- **PASSED** → "Historical criteria passed in this backtest."
- **SURVIVED** → "Not killed by the specific test shown."

These are research labels, not live-capital authorization. Historical validation does not prove forward edge.
